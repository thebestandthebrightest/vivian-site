# Vivian Glenn Personal Site

Premium minimalist personal website for Vivian Glenn, built with Next.js App Router, TypeScript, Tailwind CSS, Framer Motion, and `next/font/google`.

## Local Development

```bash
npm run dev
```

Open `http://localhost:3000`.

## Validation

```bash
npm run lint
npm run build
```

## Placeholders

- Add the final resume PDF at `public/Vivian-Glenn-Resume.pdf`.
- Update the LinkedIn placeholder URL.
- Replace the about-page travel placeholders with real travel images when available.

## Deploy

Import the repository into Vercel. The default Next.js build command, `npm run build`, is ready to use.
